Admin system and network, Solidity and JS dev

I'm 22 years old. I completed a 2-year training program in computer science specialized in systems and networks at a training center in France, and I graduated in June 2022. Afterward, I resumed programming, which I had previously set aside, and I intensified my activities starting from November 2022. On this website that I personally developed using JavaScript with React.js, I showcase my various personal programming projects as well as the two personal servers that I own and administer. The goal of this website is to continue evolving over time, and it will be regularly updated with new developments or project creations. I am passionate about blockchain, cryptocurrency, and economics. I also have a strong interest in computer science, security, and privacy. In the near future, I plan to acquire skills in smart contract development using the Solidity programming language.
